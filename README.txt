Open `index.html` in a browser. No server required.
(If your browser blocks local autoplay/preload, drop the folder into
 any static server or run `python -m http.server` from this directory.)

Brand fonts (Neue Haas Grotesk) load over the network from Adobe Fonts;
if you're offline the page falls back to a system sans-serif but everything
else — videos, reports, AHL wordmark, layout — works locally.
